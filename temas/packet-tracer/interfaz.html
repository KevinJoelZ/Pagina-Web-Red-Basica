Interfaz y Herramientas

Conoce los elementos principales del entorno de trabajo de Packet Tracer.

En este módulo

  • Introducción
  • Instalación
  • Interfaz y herramientas

Áreas principales

  • Barra de herramientas superior: Nuevo, abrir, guardar, deshacer.
  • Paleta inferior izquierda: Categorías de dispositivos (routers, switches, end devices).
  • Área de trabajo central: Lienzo donde se arma la topología.
  • Panel inferior derecho: Modo Realtime / Simulation y escala de tiempo.

Tipos de conexión

  • Copper Straight-Through: PC a switch, switch a router (FastEthernet).
  • Copper Cross-Over: Switch a switch o PC a PC (casos especiales).
  • Console: Cable de administración al router/switch.

Modos de operación

Realtime: La red funciona en tiempo real, ideal para configurar y probar ping.

Simulation: Congela el tiempo y muestra animación de paquetes — perfecto para entender el flujo de datos.

Atajo útil: Selecciona el ícono de rayo (lightning) para elegir el tipo de cable antes de conectar dos dispositivos.
