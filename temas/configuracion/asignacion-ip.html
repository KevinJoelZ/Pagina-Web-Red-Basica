Asignación de IP

Configuración de direcciones IPv4 estáticas en cada PC del laboratorio.

En este módulo

  • Introducción
  • Asignación de IP
  • Gateway y DNS

Configurar IP en un PC

  1. Haz clic en el PC → pestaña Desktop.
  2. Abre IP Configuration.
  3. Selecciona Static (configuración estática).
  4. Ingresa IP Address, Subnet Mask y Default Gateway.

Tabla de asignación del proyecto

  • PC0: 192.168.1.10 / 255.255.255.0
  • PC1: 192.168.1.11 / 255.255.255.0
  • Router (Fa0/0): 192.168.1.1 / 255.255.255.0

Configurar IP en el router

En el router, ve a Config → INTERFACE → FastEthernet0/0 e ingresa la IP y máscara. Activa el puerto (Port Status: On).
