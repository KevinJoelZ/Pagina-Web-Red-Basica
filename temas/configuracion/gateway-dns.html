Gateway y DNS

Puerta de enlace predeterminada y resolución de nombres en la red básica.

En este módulo

  • Introducción
  • Asignación de IP
  • Gateway y DNS

Default Gateway

El gateway predeterminado es la dirección IP del router en la subred local. Cuando un PC quiere comunicarse con un host fuera de su red (otra subred o Internet), envía el paquete al gateway.

En nuestro proyecto: Default Gateway = 192.168.1.1 (interfaz FastEthernet0/0 del router).

¿Qué pasa sin gateway?

Los PCs pueden hacer ping entre sí (misma subred), pero no podrán alcanzar redes externas. El gateway es obligatorio para comunicación inter-red.

Servidor DNS

El DNS traduce nombres de dominio (www.ejemplo.com) a direcciones IP. En una red básica sin servidor DNS, este campo puede dejarse vacío o apuntar a una IP simulada si se agrega un servidor al proyecto.

Verificación: Tras configurar gateway, prueba ping 192.168.1.1 desde cada PC para confirmar que alcanzan al router.
