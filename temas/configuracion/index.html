Configuración de red en Packet Tracer

Una vez armada la topología y conectados los cables, cada dispositivo necesita parámetros de red para comunicarse. Sin IP correcta, ni siquiera el ping funcionará.

Parámetros esenciales

  • IP Address: Identificador único del host en la subred.
  • Subnet Mask: Define el tamaño de la red (ej: 255.255.255.0).
  • Default Gateway: IP del router para salir de la subred local.
  • DNS Server: Resuelve nombres a direcciones IP (opcional en red básica).
