Comandos de Diagnóstico

Herramientas adicionales para inspeccionar y depurar la red básica.

En este módulo

  • Introducción
  • Pruebas con ping
  • Comandos de diagnóstico

En PCs (Command Prompt)

  • ipconfig — Muestra la configuración IP del PC.
  • arp -a — Tabla ARP: asocia IPs con direcciones MAC.
  • tracert 192.168.1.1 — Ruta que siguen los paquetes hasta el destino.

En Router (CLI)

Accede vía CLI en el router y usa:

  • show ip interface brief — Estado de todas las interfaces IP.
  • show running-config — Configuración activa del equipo.
  • show ip route — Tabla de enrutamiento del router.

Modo simulación

Activa el modo Simulation (ícono de reloj), envía un ping y observa cómo los paquetes ICMP viajan de PC → Switch → destino. Filtra por ICMP para ver solo tráfico relevante.
