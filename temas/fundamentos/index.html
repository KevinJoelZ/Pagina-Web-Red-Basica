Módulo 1

Fundamentos de Red

Base teórica para comprender cómo funciona una red local (LAN) antes de implementarla en Cisco Packet Tracer.

En este módulo

  • Introducción
  • Conceptos de red
  • Dispositivos de red
  • Modelo OSI

¿Por qué empezar por los fundamentos?

Una red básica no es solo conectar cables: implica entender cómo los dispositivos se identifican, se comunican y comparten recursos dentro de un mismo dominio de broadcast. Este módulo sienta las bases para el laboratorio práctico.

Objetivos del módulo

  • Definir qué es una red local (LAN) y sus componentes principales.
  • Identificar el rol de switches, routers y hosts en la topología.
  • Relacionar el modelo OSI con las capas involucradas en una red básica.
💡 Tip práctico: Antes de abrir Packet Tracer, dibuja en papel la topología con IPs asignadas. Esto evita errores de direccionamiento durante la configuración.
