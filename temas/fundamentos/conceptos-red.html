¿Qué es una red?

Una red de computadoras es un conjunto de dispositivos interconectados que comparten recursos e información mediante protocolos estandarizados. En nuestro proyecto nos enfocamos en una LAN (Local Area Network): red de área local que cubre un espacio reducido como un aula o laboratorio.

Red LAN vs WAN

  • LAN: Alta velocidad, bajo costo, alcance limitado (edificio, campus).
  • WAN: Conecta LANs distantes a través de proveedores (Internet es la WAN más grande).

Direccionamiento IP

Cada dispositivo en la red necesita una dirección IP única para ser identificado. En IPv4 usamos 4 octetos (ej: 192.168.1.10). La máscara de subred define qué parte es red y cuál es host (ej: 255.255.255.0 = /24).

Protocolos fundamentales

  • TCP/IP: Suite de protocolos base de Internet y redes modernas.
  • ICMP: Usado por ping para verificar conectividad.
  • ARP: Resuelve direcciones IP a direcciones MAC en la LAN.
